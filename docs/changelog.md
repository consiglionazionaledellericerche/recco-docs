# Changelog

#### Versione 1.6 ####

* Aggiunta la possibilità di notificare via email le unità di personale coinvolte nell'aggiunta o rimozione di partecipanti o Principal Investigator ad un progetto, con relativo aggiornamento dei relativi parametri di configurazione della piattaforma
* Aggiornati i parametri di configurazione della piattaforma, per supportare diversi endpoint della piattaforma [ePAS](https://epas.amministrazione.cnr.it), diverse librerie per richieste HTTP esterne
* Aggiunto il supporto a timesheet definitivi firmati sia dall'unità di personale coinvolta nel progetto, sia dal direttore

#### Versione 1.5 ####

* Migliorata la gestione delle informazioni per unità di personale non più parte dell'organico

#### Versione <1.5 ####

Non sono disponibili dettagli sulle modifiche apportate.
